使い方 / FAQ

自動モード（既定）: 見ているサイトにログインすると、その cookie が 数秒後に自動で hub の /hosts/{host} に登録され、worker がそのまま使えます。 現在開いているタブのホストだけが対象です。

開発モード: ⚙ から ON にすると自動登録を止め、cookie を 表形式で可視化・編集して手動登録できます（JSON テキスト編集は不要）。

送られるもの: name, value, domain, path, expires, secure, httpOnly, sameSite。送られないもの: 保存パスワード / Local Storage / IndexedDB （拡張 API では読めません）。完全な profile は CLI paprika-client upload-profile を使ってください。
